2.	Вывести всех актёров, которые знакомы с 2-мя и более режиссерами.
===========
select actors.name, actors.surname, count (directors.surname) as "количество режисcеров" from directors
inner join movies
on directors.director_id = movies.director_id
inner join actors_movies
USING (movie_id)
inner JOIN actors
using (actor_id)
GROUP by actors.surname
HAVING "количество режисcеров" >= 2
=============
3.	Вывести первых 10  режиссёров, которые сняли высокобюджетные фильмы. Режиссёры не должны повторяться.
===========
SELECT directors.name, directors.surname, sum ( movies.budjet) as 'Бюджет'
from movies
inner join directors
on directors.director_id = movies.director_id
GROUP by directors.surname
order by sum ( movies.budjet) DESC
limit 10
===========

6.	Вывести всех режиссёров которые снимали фильмы до 2000 года.
===========
